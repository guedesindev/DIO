Tag Font

serve específicamente para formatar a font do conteúdo. Apesar se ser mais indicado o uso do CSS esta tag tb pode ser utilizada. Utilização da tag font.

atributos:

<font color="cor">, pode ser utilizado o nome da cor, o código hexadecimal, rgb, rgba, hsl. Exemplo: Cor vermelha (red)

<font face="nome_da_fonte"> - serve para definir a tipografia da fonte. Exemplo:
Verdana,
Arial
